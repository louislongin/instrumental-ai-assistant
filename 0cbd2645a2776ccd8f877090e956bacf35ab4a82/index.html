R Setup and data import

The collected data from two studies was pre-processed. Participants were eliminated when they failed pre-registered attention and language comprehension checks (see supplementary material for detailed criteria). The subsequent data was aggregated into one csv file: “interact_long.csv”.

# load required r-packages
library(tidyverse)
library(patchwork)
library(magrittr)
library(moments)
library(ggprism)
library(lme4)

library(kableExtra)

# importing processed data as tibble
interact_long <-  read.csv("data/interact_long.csv") %>% as_tibble()

## Factorisation
interact_long$status <- factor(interact_long$status, 
                               levels=c("inactive", "active"))
interact_long$modality <- factor(interact_long$modality, 
                                 levels=c("LinguisticAI", "SensoryAI", 
                                          "SensoryTool"))
interact_long$outcome <- factor(interact_long$outcome, 
                                levels=c("negative", "positive"))
interact_long$judgment <- factor(interact_long$judgment, 
                                 levels=c("Blame/Praise", "Responsibility", 
                                          "Cause", "Counterfactual", 
                                          "Perception"))
interact_long$agent <- factor(interact_long$agent, 
                              levels=c("Driver", "AI", "tool"))
interact_long$experiment <- factor(interact_long$experiment, 
                                   levels=c("ai_exp", "tool_exp"))
interact_long$AI.Knowledge <- factor(interact_long$AI.Knowledge, 
                                     levels=c("Completely unfamiliar", 
                                              "Somewhat unfamiliar", 
                                              "Neither familiar  nor unfamiliar", 
                                              "Somewhat familiar", 
                                              "Completely familiar"))
interact_long$AI.Knowledge.2 <- factor(interact_long$AI.Knowledge.2, 
                                       levels = c("None", "A little", "Moderate", 
                                                  "Significant", "Extensive" ))
interact_long$Education <- factor(interact_long$Education, 
                                  levels = c("primary school", "high school", 
                                             "undergraduate", "graduate"))

The dataframe ‘interact_long’ is used for the subsequent analysis. It represents the experimental data in a long data format tracking each individual response in a separate row. Individual responses are individual measurements for dependent variables s.a. blame of the AI-advisor assigned by participant 1 when a crash occurs and the active AI-advisor provides sensory advice. The column ‘Response’ tracks the ratings provided by participants. The columns ‘judgment’, ‘agent’, ‘experiment’, ‘modality’, ‘outcome’, and ‘status’ further qualify the given ratings. The columns ‘AI.Knowledge’, ‘AI.Knowledge.2’, ‘Age’, ‘Education’, and ‘Gender’ track demographic variables of the respective participant. The column ‘workerId’ tracks of the respective participant using an anonymised ID.

Summary Statistics

Summary statistics of ‘interact_long’

The summary statistics include count, mean, median, standard deviation, standard error, skewness and kurtosis for each group of dependent and independent variables.

Show code 
# create summary statistics tibble
## standard statistical measures are calculated for each condition
summary_stats <- interact_long %>% 
  group_by(judgment, agent, status, experiment, outcome, modality) %>% 
  summarise(
    count = n(),
    mean = mean(Response, na.rm = TRUE) %>% round(., digits = 2),
    median = median(Response, na.rm = TRUE) %>% round(., digits = 2),
    sd = sd(Response, na.rm = TRUE) %>% round(., digits = 2),
    se = round(sd/sqrt(n()), digits = 2),
    skewness = skewness(Response, na.rm = TRUE) %>% round(., digits = 2),
    kurtosis = kurtosis(Response, na.rm = TRUE) %>% round(., digits = 2)
  )

# create summary statistics tibble which collapses 
# modality difference of AI advice

Bayesian analysis of model robustness

Load necessary Bayesian packages.

Show code 
library(brms)
library(tidybayes)
library(sjPlot)

Bayesian analysis with two models (responsibility data). Both models assume a zero-one-inflated-beta distribution for the most accurate of the model to the data. The first, simple model only calculated probabilities of the main effects as well as their interactions. The second, more complex model also calculates zoi, coi and phi for additional model insights.

Show code 
temp <- interact_long %>% 
  filter(judgment=="Responsibility",
         experiment=="ai_exp") %>%
  mutate(response_norm = (Response+100)/200)

priors_weak <-set_prior("normal(0,1)", class="b")

mod_form <- bf(
  response_norm ~ status*outcome*agent + (1|workerId),
  zoi ~ status*outcome*agent,
  coi ~ status*outcome*agent,
  phi ~ status*outcome*agent,
  family=zero_one_inflated_beta()
)
  
mod <- brm(
  formula = mod_form,
  data=temp,
  prior = priors_weak,
  chains=4,
  iter=6000,
  cores=,
  file="data/mod_zoib_priors_phi_coi_zoi_responsibility"
)

mod_form_simple <- bf(
  response_norm ~ status*outcome*agent + (1|workerId),
  family=zero_one_inflated_beta()
)

mod_simple <- brm(
  formula = mod_form_simple,
  data=temp,
  prior = priors_weak,
  chains=4,
  iter=6000,
  cores=,
  file="data/mod_zoib_priors_phi_coi_zoi_responsibility_simple"
)
